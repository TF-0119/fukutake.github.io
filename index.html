匿名プログラミングコミュニティ

毎週ちょっとした「お題」を出して、みんなで作って見せ合うだけのサイト。

やりたいこと（ざっくり）

  • 毎週お題（例：好きな飲み物のLPを作る / APIで天気を出す）
  • 作ったらURLを貼る → みんなで見て評価　(高評価/コメント)
  • 投票でゆるいランキング。称号がちょっと貯まる
  • 困ったらミニQ&A（知恵袋っぽく、匿名でOK）

トーン & ルール

  • ゆるい / 褒め合い重視 🙌
  • 匿名でOK、実名強要なし
  • パクリ禁止・引用は明記

なにが楽しい？

  • 毎週「作る理由」ができる
  • 人の作品から学べる
  • 承認欲求が満たせる

こんなお題を想定

  • 初級：自己紹介ページ / フォーム作成 / 画像ギャラリー
  • 中級：外部APIで何か出す / レスポンシブ対応
  • 上級：小さなWebアプリ / ミニゲーム / パフォーマンス改善
